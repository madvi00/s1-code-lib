이런 점이 좋아요

✓
설치·빌드 없이 즉시 사용
순수 HTML/CSS/JS로 구성. 어떤 프로젝트에도 폴더만 복사하면 바로 적용됩니다.
✓
복사 한 번이면 끝나는 스니펫
각 컴포넌트 페이지에 미리보기와 HTML 코드 스니펫이 나란히 있어, 바로 복사해 프로젝트에 붙여 넣을 수 있습니다.
✓
디자인 토큰 기반의 일관된 스타일
Foundation · Semantic 2단계 토큰 구조로 Color, Typography, Spacing, Sizing, Radius, Border가 통일됩니다.
✓
퍼블리싱 가이드 · 사용 가이드라인 내장
전사 공통 퍼블리싱 표준(인코딩·접근성·코드 규칙)과 컴포넌트별 Do's & Don'ts를 사이트 내에서 바로 확인할 수 있습니다.

사용 방법

1
 전달받은 폴더 전체를 프로젝트 루트에 그대로 복사
2
 index.html 을 브라우저로 열면 라이브러리 사이트 실행
3
 사이드바에서 원하는 컴포넌트를 선택 → 미리보기 옆 코드 스니펫을 복사해 사용
문의 / 피드백
